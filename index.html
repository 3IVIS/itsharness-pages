itsharness is an open-source visual builder for production AI harnesses: canvas, neutral spec, safety rails, run loop. One graph editor, one runtime-neutral IR, adapters for LangGraph, CrewAI, Mastra, and (planned) Microsoft Agent Framework — plus A2A, the protocol layer that makes itsharness the orchestrator of orchestrators.

§ 02 · specification

FlowSpec v0.2.0. The IR is live. The canvas authors it.

Two required fields, fourteen node types, a discriminated edge union, three optional config sections. The canvas emits this on every change. Adapters consume it. Below — a real flow, in real JSON.

§ 05 · v0.4 pass28 status

Three runtimes. Hardened. Traced.

All 14 node types are drawable, configurable, validated, and — for LangGraph and CrewAI — executable end-to-end. HITL pause/resume works. Every run produces a Langfuse trace with per-node child spans. v0.4 (pass28) closed a 12-fix security and hardening pass on top.

SECURITY & HARDENING
Critical: /run now validates fn_ref before exec().

POST /run previously skipped validate_spec(), so path-traversal and shell-injection fn_ref values reached exec() directly when callers bypassed /compile. Validation now runs at all three entry points — /compile, /flows, and /run. Two regression tests added. Plus: timing-safe login, 1 MB body cap, CSP header, password policy, proxy-aware rate limiter, JOB_TTL_HOURS eviction, separate LiteLLM database, non-root containers.
